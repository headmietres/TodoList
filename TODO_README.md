# TodoList — 桌面端待办记录小工具

轻量级、常驻桌面的待办管理工具，基于 **PDCA（计划-执行-检查-处理）** 循环模型，以日历视图为核心规划入口，结合子任务拆解、环形进度可视化与游戏化激励系统。

## 快速开始

```bash
cd TodoList
chmod +x start.command
./start.command
```

或者：

```bash
npx electron .
```

## 系统要求

- macOS 12+ / Windows 10+ / Linux
- Node.js 18+
- 屏幕分辨率 1280×800 及以上

## 核心功能

### 日历视图

标准月历网格，每个日期格根据任务完成率显示不同底色（绿/黄/红梯度）。点击任意日期查看当日任务列表。

- 上月/下月切换，一键回到今天
- 完美日（当日全部完成）自动标记
- 任务完成率颜色直观反馈

### 任务管理

- **添加任务** — 输入标题 + 选择优先级 + 选择标签，支持回车提交
- **优先级** — 高（红）/ 中（黄）/ 低（绿）三级
- **标签** — 自定义创建标签，可设颜色，任务关联一个标签
- **截止日期** — 可选，点击 ⋯ 展开日期选择器
- **子任务** — 每个任务可拆分子任务，支持 ＋ 添加、勾选、× 删除，默认收起

### 环形进度

- **任务进度小环** — 基于子任务完成度实时计算百分比
- **今日总进度大环** — 今日任务整体完成率一目了然

### 统计看板

- 总体完成率
- 已完成/总数统计
- 经验等级展示
- 近 7 天完成趋势条形图
- 子任务总体完成率
- 完美日天数

### 激励系统

- **经验值（XP）** — 完成任务获得经验，主任务 +10 XP，子任务 +2 XP
- **等级系统** — 累积经验提升效率等级，升级有庆祝动画
- **完美日成就** — 当日所有任务完成触发庆祝提示，额外 +20 XP

### 迷你模式

窗口可折叠为紧凑视图，仅显示日期、添加任务栏和今日任务列表，适合常驻桌面角落。

### 任务归档

已完成/过期任务可移入归档区，支持一键恢复。归档视图集中管理所有已归档任务。

### 设置面板

- 浅色/深色/跟随系统 主题切换
- 窗口置顶
- 提醒音效开关
- 每日任务限额
- 标签管理（新增/删除）

### 撤销/重做

支持 `Ctrl+Z` / `Ctrl+Shift+Z` 撤销和重做任务操作。

### 全局快捷键

`Ctrl+Shift+T` 快速呼出/隐藏主窗口。

## 数据存储

所有数据纯本地存储于系统用户数据目录：

- **macOS**: `~/Library/Application Support/TodoList/data/`
- **Windows**: `%APPDATA%/TodoList/data/`
- **Linux**: `~/.config/TodoList/data/`

存储格式为 JSON 文件，不联网上传，无需注册登录。

## 项目结构

```
TodoList/
├── start.command          # macOS 一键启动脚本
├── package.json
├── assets/
│   └── icon.png           # 应用图标
└── src/
    ├── main/
    │   ├── main.js        # Electron 主进程（窗口/托盘/IPC/快捷键）
    │   └── storage.js     # 数据持久化（JSON 文件读写/撤销/等级）
    ├── preload.js          # 安全桥接（contextBridge API）
    └── renderer/
        ├── index.html      # 界面结构
        ├── app.js          # 全部业务逻辑（日历/任务/统计/设置）
        └── styles/
            └── main.css    # 样式表（Flat Design + Inter 字体）
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 提交添加任务 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+F` | 聚焦到任务输入框 |
| `Ctrl+Shift+T` | 全局呼出/隐藏窗口 |

## 技术栈

- **框架**: Electron 28
- **语言**: JavaScript (CommonJS)
- **渲染**: 原生 DOM API
- **存储**: 本地 JSON 文件
- **依赖**: 无第三方 UI 库，纯手写
